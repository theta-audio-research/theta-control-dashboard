# THETA Control Room — v3.2 Test Tickets + Return Packets

## Built

v3.2 adds a lightweight test-ticket system into each project detail view.

## Key features

- 5–10 project test questions seeded per plugin/project.
- Clickable pass checkbox.
- Status dropdown: open / passed / failed / blocked.
- Tester and version fields.
- Notes textbox.
- Completed/archive section.
- Return packet generator for project chats.

## API endpoints

- `POST /api/ticket`
- `POST /api/ticket/complete`
- `POST /api/return-packet`
