# THETA Control Room — v3.0 Automation Build

## Built

v3.0 Delta Inbox Automation adds a Cloudflare Worker API and scheduled checkpoint system.

## Timing

The Worker is configured for these local checkpoint times:

| Local time | Purpose |
|---|---|
| 6 AM | Early planning checkpoint |
| 9 AM | Morning work checkpoint |
| 12 PM | Midday checkpoint |
| 2 PM | Afternoon redirect checkpoint |
| 4 PM | Late-day checkpoint |
| 11 PM | End-of-night checkpoint |

Timezone: America/Los_Angeles.

## Automation model

| Layer | Automated? |
|---|---|
| GitHub commit → Cloudflare redeploy | Yes |
| Delta JSON → projects.json/report update | Yes, after Worker setup |
| Scheduled checkpoint reports | Yes, after Worker setup |
| Plugin build → delta event | Yes, once hook is added to build scripts |
| ChatGPT/Claude/Gemini chats → automatic delta | Not by default; they must emit/send delta JSON |
